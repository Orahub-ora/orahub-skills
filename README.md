# orahub-skills

This repository is prepared as a bundled OraHub skills package.

## Structure

- `SKILL.md`: repository-level package descriptor
- `photo-color-match/SKILL.md`: color matching skill
- `photo-passersby-removal/SKILL.md`: passersby removal skill
- `photo-remove-background/SKILL.md`: background removal skill for transparent-background output
- `photo-background-replace/SKILL.md`: background replacement skill
- `references/platform-compatibility.md`: shared platform compatibility and OraHub reference
